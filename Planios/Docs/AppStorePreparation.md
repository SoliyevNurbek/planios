# Planios App Store Preparation

## App Name Ideas

- Planios
- FocusMint
- GreenStreak
- DayForge
- NextBlock

## App Icon Concept

Use a deep green rounded-square background with a white checkmark seal or target ring. Add a subtle mint highlight in the top-left corner to make the icon feel premium without becoming busy.

## Launch Screen Design

Use a static green background with the centered app mark and the wordmark `Planios` below it. Keep the launch screen minimal and let the animated SwiftUI splash provide the premium transition once the app loads.

## App Store Description

Planios helps you plan the day, protect focus, and finish what matters.

Create tasks for today, tomorrow, or the week ahead. Schedule reminders before work starts, run focused countdown sessions, and track completion, streaks, and weekly performance in a clean interface designed to reduce friction.

## Keywords

productivity, planner, daily planner, task manager, focus timer, discipline, streak tracker, reminders, weekly planner, habits
